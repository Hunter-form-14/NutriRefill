# NutriRefill

主著者として『NutriSynth: 栄養素を添加する「栄養メディア」の提案とパーソナライズされた補完への応用』という題で、アサヒグループと共同で研究開発に取り組み、エンタテインメントコンピューティング2025でデモ・ポスター発表しました。背景として、従来の味覚メディアは味をデジタルに表現できますが栄養を供給できず、一方サプリメント等は食体験と分離され継続が困難であるという課題がありました。そこで、味覚メディアと連動して栄養を補完する「栄養メディア」という新概念を提案し、システム開発を主導しました。具体的には、カルシウムや鉄、ビタミン類など8種類の栄養素をμL単位で吐出できるデバイス「NutriSynth」を設計・実装しました。栄養素の劣化抑制のため冷蔵・遮光機構を組み込み、スターラーによる均一撹拌やpH調整で保存性も確保しています。さらに、GPT-4とRAGを活用して食事画像から栄養素を推定し、ユーザーの年齢・性別に基づいた不足分を可視化し、「NutriSynth」から吐出させるWebアプリ「NutriRefill」も開発しました。応用事例として、本システムにより、罪悪感なく食べられる豚骨風ラーメンとカスタードクリーム風スイーツを開発し、満足感と健康の両立を達成しました。本研究は食体験を制約から解放し、日常生活から宇宙空間まで応用可能なパーソナライズされた栄養管理の新形態を提示しました。
